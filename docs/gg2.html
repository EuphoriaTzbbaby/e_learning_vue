3. 实体关系定义

用户 → 视频合集 (1:N)

一个用户(教师)可以创建多个视频合集，每个视频合集只属于一个用户。

外键: Album.teacher_id → User.id

视频合集 → 视频 (1:N)

一个视频合集可以包含多个视频，每个视频只属于一个视频合集。

外键: Video.album_id → Album.id

用户 → 英语学习 (1:N)

一个用户可以有多个英语学习记录，每个英语学习记录只属于一个用户。

外键: English.user_id → User.id

英语学习 → 复习状态 (1:1)

一个英语学习记录对应一个复习状态，一个复习状态只对应一个英语学习记录。

外键: ReviewState.eg_id → English.id

用户 → 复习记录 (1:N)

一个用户可以有多个复习记录，每个复习记录只属于一个用户。

外键: ReviewLog.user_id → User.id

英语学习 → 复习记录 (1:N)

一个英语学习记录可以有多个复习记录，每个复习记录只对应一个英语学习记录。

外键: ReviewLog.eg_id → English.id

用户 → AI阅读理解 (1:N)

一个用户可以有多个AI阅读理解记录，每个AI阅读理解记录只属于一个用户。

外键: AIReading.user_id → User.id
