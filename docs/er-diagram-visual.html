英语学习系统E-R图 - 可视化版本

实体详细定义

用户 (User)
属性名 代码 数据类型 约束 说明
用户ID id BIGINT 主键, 自增 用户唯一标识符
用户名 username VARCHAR(50) 非空, 唯一 用户登录名
密码 password VARCHAR(100) 非空 加密存储的密码
邮箱 email VARCHAR(100) 唯一 用户邮箱地址
角色 role VARCHAR(20) 非空 用户角色: STUDENT/TEACHER/ADMIN
是否删除 is_deleted TINYINT 默认值: 0 逻辑删除标记
视频合集 (Album)
属性名 代码 数据类型 约束 说明
合集ID id BIGINT 主键, 自增 合集唯一标识符
合集标题 title VARCHAR(100) 非空 合集名称
教师ID teacher_id BIGINT 非空, 外键 关联User表的id
封面URL cover_url VARCHAR(255) 合集封面图片URL
是否删除 is_deleted TINYINT 默认值: 0 逻辑删除标记
视频 (Video)
属性名 代码 数据类型 约束 说明
视频ID id BIGINT 主键, 自增 视频唯一标识符
合集ID album_id BIGINT 非空, 外键 关联Album表的id
视频标题 title VARCHAR(100) 非空 视频名称
视频URL video_url VARCHAR(255) 非空 视频文件URL
封面URL cover_url VARCHAR(255) 视频封面图片URL
是否删除 is_deleted TINYINT 默认值: 0 逻辑删除标记
英语学习 (English)
属性名 代码 数据类型 约束 说明
英语ID id BIGINT 主键, 自增 英语学习记录唯一标识符
用户ID user_id BIGINT 非空, 外键 关联User表的id
英语单词 word VARCHAR(100) 非空 英语单词
中文翻译 translation VARCHAR(255) 非空 单词中文翻译
难度级别 level INT 单词难度等级
是否删除 is_deleted TINYINT 默认值: 0 逻辑删除标记
复习状态 (ReviewState)
属性名 代码 数据类型 约束 说明
状态ID id BIGINT 主键, 自增 复习状态唯一标识符
英语ID eg_id BIGINT 非空, 外键, 唯一 关联English表的id
复习次数 review_count INT 默认值: 0 已复习次数
下次复习时间 next_review_time DATETIME 计划下次复习时间
熟悉度 familiarity INT 默认值: 0 单词熟悉程度
是否删除 is_deleted TINYINT 默认值: 0 逻辑删除标记
复习记录 (ReviewLog)
属性名 代码 数据类型 约束 说明
记录ID id BIGINT 主键, 自增 复习记录唯一标识符
英语ID eg_id BIGINT 非空, 外键 关联English表的id
用户ID user_id BIGINT 非空, 外键 关联User表的id
复习时间 review_time DATETIME 非空 复习的时间
复习结果 result INT 复习结果评分
是否删除 is_deleted TINYINT 默认值: 0 逻辑删除标记
AI阅读理解 (AIReading)
属性名 代码 数据类型 约束 说明
阅读ID id BIGINT 主键, 自增 阅读记录唯一标识符
用户ID user_id BIGINT 非空, 外键 关联User表的id
阅读内容 content TEXT 非空 阅读文章内容
问题 question TEXT AI生成的问题
回答 answer TEXT 用户的回答
评分 score INT AI评分结果
阅读时间 read_time DATETIME 非空 阅读的时间
是否删除 is_deleted TINYINT 默认值: 0 逻辑删除标记
